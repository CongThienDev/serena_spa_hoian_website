# Claude Code Start Prompt

Use this prompt with Claude Code:

```txt
You are building the Serena Spa Hội An website.
Read all markdown files inside /docs first before coding.

Goal: Build a premium wellness spa website using Next.js App Router, TypeScript, Tailwind CSS, Framer Motion, and Sanity CMS-ready architecture.

Brand direction:
- Warm cream, peach, terracotta, dark brown
- Luxury wellness, minimal zen, Japandi spa
- Heading font: Cormorant Garamond or Playfair Display
- Body font: Inter or Manrope
- Rounded cards, soft shadows, subtle animations

Must build MVP pages:
- Home
- About
- Services listing
- Service detail template
- Wellness page/section
- Gallery
- Blog listing
- Blog detail template
- Booking
- Contact
- Policy pages

Must include:
- Responsive mobile-first design
- Sticky mobile CTA: Call, WhatsApp/Zalo, Book Now
- Desktop floating contact widget
- Services and Wellness hover mega menu
- Service cards with hover states
- Booking form with validation
- SEO-ready metadata
- JSON-LD schema placeholders

Start by creating project structure, Tailwind theme tokens, layout components, and homepage.
Do not overcomplicate with payment/login/CRM in MVP.
```
