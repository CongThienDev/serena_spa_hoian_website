# Mobile UX Specs

## Priority
Khách spa/tourist chủ yếu dùng mobile. Mobile phải nhanh hơn desktop.

## Mobile header
- Logo left
- Book button or phone icon
- Hamburger menu right
- Sticky header on scroll

## Mobile hero
- Image top or background
- H1 max 2–3 lines
- CTA stack vertical
- avoid too much text

## Sticky bottom bar
Always visible:
- Call
- Zalo/WhatsApp
- Book Now

## Floating widget mobile
Desktop floating widget có thể chuyển thành bottom bar để không che nội dung.

## Services mobile
- Category chips horizontal scroll
- Cards 1 column
- CTA full width
- Service detail uses sticky bottom Book Now

## Booking mobile
- One-column form
- Large tap targets
- Date/time input easy to use
- Submit button sticky at bottom after fields visible

## Mobile typography
- H1 40–48px
- H2 30–36px
- Body 15–16px
- Buttons minimum height 44px

## Performance mobile
- Use next/image
- Lazy load below fold
- compress images to webp/avif
- avoid heavy animation
