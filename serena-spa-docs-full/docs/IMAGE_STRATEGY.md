# IMAGE_STRATEGY.md — Image Usage Plan

## Existing Assets
The user provided:
- Business card/front-back design
- Serena Spa brand card
- Interior treatment room images
- Bathroom/detail interior images

## Visual Strength
The interiors already support a premium warm wellness website. Use real spa images prominently.

## Recommended Aspect Ratios
- Hero: 16:9 or 21:9
- Service card: 4:3 or 16:10
- Gallery masonry: mixed 4:5, 1:1, 16:10
- Blog thumbnail: 16:10
- Open Graph: 1200x630

## Image Naming
Use SEO-friendly names:
- `serena-spa-hoi-an-treatment-room.jpg`
- `serena-spa-hoi-an-relaxing-massage-room.jpg`
- `serena-spa-hoi-an-wellness-interior.jpg`

## Optimization
- Convert to WebP/AVIF.
- Compress photos.
- Use responsive sizes.
- Blur placeholder optional.

## Homepage Image Usage
- Hero: best wide spa room image.
- Services: treatment-related crop.
- About: interior/wash area.
- Gallery: all real photos.

## Avoid
- Overusing generic stock images.
- Using low-resolution business card images as hero.
- Text embedded inside photos for SEO content.
