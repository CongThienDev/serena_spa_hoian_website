This is a luxury wellness hospitality experience.

## Tech Stack
- Next.js App Router
- TypeScript
- Tailwind CSS
- Framer Motion
- Sanity CMS
- Vercel

## Responsibilities
- scalable architecture
- reusable components
- clean code
- responsive layouts
- accessibility
- performance optimization

## Coding Rules
- mobile first
- server components by default
- avoid client components unless necessary
- avoid duplicated logic
- avoid inline styles
- use semantic HTML
- optimize all images
- maintain spacing consistency

## Design Philosophy
- spacious
- calm
- editorial
- cinematic
- minimal luxury

## Avoid
- generic startup UI
- flashy animations
- crowded layouts
- unnecessary dependencies
- hardcoded values
