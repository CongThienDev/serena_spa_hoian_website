# Customer Journey / User Flow

## Flow 1: Social visitor
```txt
Instagram / Facebook / Naver
→ Home
→ Gallery / Services
→ Service Detail
→ Book Now / WhatsApp
→ Staff confirms booking
```

## Flow 2: Google SEO visitor
```txt
Google search: best spa in Hoi An
→ SEO landing page or blog article
→ Related service cards
→ Service detail
→ Booking form
→ Confirmation
```

## Flow 3: Returning customer
```txt
Direct website visit
→ Booking
→ Choose previous/favorite service
→ Submit phone/name
→ Confirm via Zalo/WhatsApp
```

## Flow 4: Tourist urgent booking
```txt
Mobile search
→ Home
→ Floating WhatsApp/Call
→ Direct chat/call
```

## Flow 5: Korean customer
```txt
Naver Blog / Google
→ Korean landing/blog page
→ KakaoTalk CTA
→ Booking confirmation
```
