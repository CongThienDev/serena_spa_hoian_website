# ACCESSIBILITY.md — Accessibility Requirements

## Semantic HTML
- Header, nav, main, section, footer.
- Use buttons for actions and links for navigation.

## Keyboard
- All menus accessible by keyboard.
- Visible focus states.
- Modal traps focus.
- Esc closes modal/dropdown.

## Colors
- Ensure text contrast is readable.
- Do not place light text on light peach without contrast.

## Images
- Decorative images can use empty alt.
- Informational images need descriptive alt.

## Forms
- Every input has label.
- Errors are visible and announced.
- Required fields clearly marked.

## Motion
- Respect prefers-reduced-motion.

## Mobile
- Tap targets at least 44px.
- Floating contact widget must not block booking button.
