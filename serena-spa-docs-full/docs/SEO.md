# SEO.md — Search Engine Optimization Blueprint

## Main SEO Goal
Rank for spa and massage-related searches in Hoi An and convert search visitors into bookings.

## Primary Keywords
- spa hoi an
- best spa in hoi an
- massage hoi an
- wellness spa hoi an
- foot massage hoi an
- facial care hoi an
- couple massage hoi an
- body massage hoi an
- hoi an spa near old town

## URL Rules
Use clean, lowercase, hyphenated URLs:

```txt
/services
/services/massage
/services/facial-care
/services/serena-signature-massage
/wellness/detox-and-revive
/blog/best-spa-in-hoi-an
/contact
```

## Metadata Pattern
### Home
Title: `Serena Spa Hoi An | Premium Relaxation & Wellness Spa`
Description: `Experience premium massage, facial care, body therapy and wellness treatments at Serena Spa Hoi An. Book your relaxing escape today.`

### Service Detail
Title: `[Service Name] in Hoi An | Serena Spa Hoi An`
Description: `[Service Name] with natural oils, professional therapists and a peaceful spa space in Hoi An. View duration, price and book now.`

### Blog
Title: `[Article Title] | Serena Spa Hoi An Blog`
Description: 140–160 character summary with target keyword.

## Heading Hierarchy
- One H1 per page.
- H2 for major sections.
- H3 for cards/FAQ items.
- Do not use headings only for styling.

## Schema Requirements
Implement:
- LocalBusiness / HealthAndBeautyBusiness / Spa
- Service schema for service detail pages
- Product schema for products if added
- Article schema for blog posts
- FAQPage schema for FAQ blocks
- BreadcrumbList schema for nested pages

## Internal Linking
Every blog should link to:
- Relevant service page
- Booking page
- Contact page

Every service detail should link to:
- Parent category
- Related services
- Booking page

## Image SEO
- Use descriptive filenames.
- Add alt text with natural context.
- Avoid keyword stuffing.

Example:
`serena-spa-hoi-an-relaxing-massage-room.jpg`
Alt: `Warm treatment room at Serena Spa Hoi An with massage beds and soft lighting.`
