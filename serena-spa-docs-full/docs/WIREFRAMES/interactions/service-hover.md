# Interaction — Services Hover/Menu

## Desktop
Hover Services nav:
- Open rounded mega menu.
- Left column: categories.
- Hover category: show treatment list.
- Right card: image and CTA.

## Mobile
Tap Services:
- Expand accordion.
- Tap category to reveal treatments.
- Include Book Now button.

## Animation
- Fade + slide down 12px.
- 220–320ms.
- Close on outside click/Escape.
