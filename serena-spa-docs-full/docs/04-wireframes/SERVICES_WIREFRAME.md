# Services Page Wireframe

## Desktop layout

```txt
Top bar + header

Page hero
- title: Services & Treatments
- subtitle
- category tabs

Service category cards
- Massage
- Facial Care
- Body Therapy
- Wellness
- Packages & Combos

Services grid
- 3 or 4 columns desktop
- card content:
  - image
  - category label
  - service name
  - short description
  - duration
  - price
  - View Details
  - Book Now

Trust strip

FAQ section

CTA banner

Footer
```

## Hover behavior
Service card hover:
- image zoom 1.05
- soft overlay
- reveal two buttons:
  - View Details
  - Quick Book
- border turns terracotta
- shadow increases slightly

## Filter behavior
- Tabs filter by category without page reload.
- URL can update query: `/services?category=massage`.

## Dropdown behavior from nav
When hovering `Services`:

Level 1 menu:
- Massage
- Facial Care
- Body Therapy
- Foot & Hand Care
- Wellness Programs
- Packages & Combos
- All Treatments

When hovering `Massage`, show level 2:
- Serena Signature Massage
- Aromatherapy Massage
- Thai Massage
- Hot Stone Therapy
- Foot Massage
- Herbal Compress
- View all Massage
