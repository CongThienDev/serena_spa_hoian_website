# Serena Spa Hội An Website Blueprint

Bộ tài liệu này dùng để đưa cho Claude Code hoặc developer trước khi bắt đầu code website Serena Spa Hội An.

Mục tiêu: xây dựng website spa/wellness premium, tối ưu mobile, SEO, booking/contact nhanh, đa ngôn ngữ, dễ mở rộng blog và dịch vụ.

## Cấu trúc docs

- `00-overview`: tóm tắt dự án, MVP, scope
- `01-brand`: màu sắc, font, visual direction
- `02-content`: content cần chuẩn bị
- `03-sitemap-flow`: sitemap, customer journey, full flow
- `04-wireframes`: wireframe desktop từng trang
- `05-mobile`: mobile UX
- `06-seo`: SEO architecture, keywords, schema
- `07-tech-stack`: stack code đề xuất
- `08-components`: component specs
- `09-booking`: booking/contact flow
- `10-roadmap`: roadmap, phases thực thi
- `11-progress`: progress tracker
- `12-claude-code-prompts`: prompt dùng với Claude Code
- `assets-references`: ghi chú asset/hình ảnh
